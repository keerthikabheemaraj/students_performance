# students_performance_analysis

## Project Overview

This project analyzes student performance data to identify trends, patterns, and factors affecting academic scores. The analysis includes data cleaning, exploratory data analysis (EDA), statistical analysis, visualizations, and an interactive dashboard.

## Project Objective

The objective of this project is to:

* Analyze student performance data.
* Identify factors influencing academic scores.
* Perform data cleaning and preprocessing.
* Conduct exploratory data analysis (EDA).
* Generate meaningful visualizations.
* Perform statistical analysis.
* Build an interactive dashboard.
* Provide actionable insights and recommendations.

## Dataset

**Dataset Name:** Students Performance in Exams

The dataset contains information about students, including:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

## Data Cleaning

The following preprocessing steps were performed:

1. Checked for missing values.
2. Removed duplicate records.
3. Standardized column names.
4. Verified data types.
5. Prepared the dataset for analysis.

## Exploratory Data Analysis (EDA)

The following analyses were performed:

* Score distribution analysis.
* Gender-wise performance comparison.
* Test preparation course impact analysis.
* Parental education impact analysis.
* Subject-wise performance comparison.

## Statistical Analysis

Statistical measures calculated include:

* Mean
* Median
* Standard Deviation
* Minimum and Maximum Values
* Correlation Matrix

## Visualizations

The project includes:

* Bar Charts
* Histograms
* Pie Charts
* Correlation Heatmaps
* Comparative Performance Charts

## Dashboard Features

The Streamlit dashboard provides:

* Dataset Preview
* Summary Statistics
* Interactive Visualizations
* Performance Insights
* User-Friendly Interface

## Key Insights

1. Students who completed the test preparation course generally achieved higher scores.
2. Reading and writing scores show a strong positive correlation.
3. Female students tend to perform better in reading and writing.
4. Male students slightly outperform in mathematics.
5. Higher parental education levels are associated with better student performance.

## Recommendations

* Encourage participation in test preparation programs.
* Provide additional support for students with lower scores.
* Increase parental involvement in academic activities.
* Develop targeted learning strategies for weaker subjects.
